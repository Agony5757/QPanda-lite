# State Preparation Examples

量子态制备示例代码。

## 示例列表

| File | Description |
|------|-------------|
| [hadamard_superposition.py](hadamard_superposition.py) | 使用 Hadamard 门创建均匀叠加态 |
| [rotation_prepare.py](rotation_prepare.py) | 通过旋转制备任意态（Bell、GHZ、W、随机态） |

## 运行方式

从仓库根目录执行：

```bash
python examples/state_preparation/hadamard_superposition.py
python examples/state_preparation/rotation_prepare.py
```
